FREQUENTLY ASKED QUESTIONS

  • Μπορώ να γίνω μέλος της τοπικής επιτροπής της Ξάνθης;
    Ναι, αν είσαι προπτυχιακός ή μεταπτυχιακός φοιτητής του τμήματος Ηλεκτρολόγων Μηχανικών και Μηχανικών Υπολογιστών του ΔΠθ.

  • Πώς μπορώ να εγγραφώ;
    Το μόνο που χρειάζεται είναι να συμπληρώσεις τη φόρμα στο Join Us.

  • Πρέπει να πληρώσω για την εγγραφή
    Όχι, η εγγραφή είναι τελείως δωρεάν.

  • Τι πρέπει να κάνω μετά την εγγραφή μου;
    Η οργάνωση μας είναι εθελοντική, επομένως δεν χρειάζεται να κάνεις κάτι μετα την εγγραφή σου.
    • Ωστόσο μπορείς να επικοινωνήσεις μαζί μας ώστε να συμμετάσχεις σε μια από τις ομάδες μας, οι οποίες καλύπτουν ένα ευρύ φάσμα δραστηριοτήτων, όπως:
    IT Team - Συντήρηση & δημιουργία περιεχομένου για το site (eestec.ee.duth.gr)
    Design Team - Υπεύθυνη για τη δημιουργία & σχεδίαση οποιουδήποτε προωθητικού υλικού.
    PR Team - Υπεύθυνη για την προώθηση του οργανισμού μας τόσο σε τοπικό όσο και σε διθνές επίπεδο.
    FR Team - Υπεύθυνη για πόρων για τη σωστή λειτουργία των τοπικών αλλά και των διεθνών δράσεων του τοπικού παραρτήματος.

  • Πώς μπορώ να συμμετέχω σε ενα event;
    Θα βρεις όλες τις πληροφορίες που χρειάζεσαι στο Join an Event.

  • Τα event είναι δωρεάν;
    Ναι, με εξαίρεση κάποια που στην επίσημη ανακοίνωση για τις αιτήσεις συμμετοχής αναγράφουν το ακριβές πόσο, το οποίο σου ζητείται. Το ποσό αυτό, ωστόσο, είναι μικρό. Με το ποσό αυτό καλύπτονται όλα σας τα έξοδα διαμονής, διατροφής και τοπικών μεταφορών.

  • Ποια έξοδα μου είναι καλυμένα σε ένα διεθνές event;
    Σε όλα τα διεθνή event όλα τα έξοδα διαμονής και διατροφής καλύπτονται απο την εκάστοτε οργανωτική τοπική επιτροπή.

  • Μπορώ να συμμετέχω σε ενα διεθνές event που οργανώνεται στην Ξάνθη αν είμαι μέλος της τοπικής επιτροπής;
  • Στα διεθνή event που οργανώνονται στην Ξάνθη μπορείτε να συμμετέχετε σαν διοργανωτής. Ωστόσο, αρκετά διεθνή event που διοργανώνουμε παρέχουν τη δυνατότητα συμμετοχής σαν τοπικός participant.

  • Μπορώ να συμμετέχω σε ενα τοπικό event που οργανώνεται στην Ξάνθη αν είμαι μέλος της τοπικής επιτροπής;
  • Φυσικά, κάθε μέλος μπορεί να συμμετέχει στο τοπικά event που διοργανώνονται.

  • Μπορούν άτομα που δεν είναι μέλη να συμμετέχουν σε τοπικά event;
  • Ναι, τα περισσότερα τοπικά event είναι ανοιχτά για όλους τους φοιτητές του τμήματος ΗΜΜΥ του ΔΠΘ και για φοιτητές από άλλα τμήματα που ενδιαφέρονται να συμμετέχουν. Ωστόσο, ίσως πραγματοποιηθούν κάποια events, όπως εξειδικευμένα trainings, που θα απευθύνονται μόνο σε μέλη.

  • Ποιες είναι οι δράσεις της EESTEC LC Xanthi;
  • Μπορείς να ενημερωθείς για τις τοπικές δράσεις μας στο Local events και για τις διεθνείς δράσεις στο International events.

  • Πώς μπορώ να επικοινωνήσω μαζί σας;

    - Μέσω της σελίδας στο facebook
    - Είτε μέσω της φόρμας που θα βρείτε εδώ.
